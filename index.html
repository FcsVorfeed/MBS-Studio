Research

The Covariant Emergence Principle (CEP)

This project introduces the "Covariant Emergence Principle" (CEP), a heuristic ontological razor for fundamental physics.

The core argument is based on an information-theoretic criterion: "Coordination implies Emergence." We propose that any physical quantity (such as spacetime metric or relativistic mass) that requires intricate, synchronized transformations (covariance) to maintain consistency is, by definition, not a fundamental entity but a macroscopic effective description similar to thermodynamic temperature.
